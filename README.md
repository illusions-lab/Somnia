# Somnia

Somnia is a family of Japanese language models fine-tuned from
[sonoisa/t5-japanese](https://github.com/sonoisa/t5-japanese).

Each generation grows in scale and capability, named after stages of a dream:

| Model     | Meaning        |
|-----------|----------------|
| Doze      | うたた寝 — the first, smallest model |
| Dream     | 夢 |
| Reverie   | 夢想 |
| Vision    | 幻視 |
| Legend    | 伝説 |
